# FINESSE Basics

This section contains introductory FINESSE simulations and modelling exercises.

Topics covered include:

- Power splitting at partially reflective mirrors
- Optical field amplitude and phase analysis
- Effect of mirror tuning phase and propagation length
- Fabry–Perot cavity resonance behaviour
- Free Spectral Range (FSR), FWHM, and cavity finesse
- Michelson interferometer response and fringe behaviour
- Gravitational-wave transfer function simulations
- Pound–Drever–Hall (PDH) error signal generation
- Optical cavity locking and feedback concepts
